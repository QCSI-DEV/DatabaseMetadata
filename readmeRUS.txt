ФУНКЦИОНАЛ
Данный коллектор метаинформации баз данных предоставляет метаинформацию о структуре баз данных(SQLite и Postgres):
- о существующих таблицах базы
- о принадлежащих таблицам полях , их типах данных, значении по умолчанию.
- о значении главных ключей таблиц.
- о внешних ключах таблицы (влючая зависимые таблицы и поля)
- о созданных индексах, их псевдониме , о полях входящих в данный индекс, о уникальности индекса.  


НАСТРОЙКИ:
Точка запуска проекта: com.qsci.database.metadata.Run.
Базы данных конфигурируються в resources/data.properties следующим образом:
-для SQLITE баз данных:

	1 Указать в поле "url" путь к базе данных.
	2.Значение логина("login") и пароля("password") следует установить как null.
		ПРИМЕР:
		url = jdbc:sqlite:mySQLiteBASE.sl3
		login = null
		password = null

-для POSTGRES баз данных:
	 
	1.указать путь к базе данных.
	2.установить пароль.
	3.установить логин.
		ПРИМЕР: 
		url = jdbc:sqlite:myPostgresBASE.sl3
		login = postgresql
		password = password


БАЗА ДАННЫХ ПО УМОЛЧАНИЮ: SQLITE Northwind.sl3.

В планах добавить поддержку mySQL и DERBY.
